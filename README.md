lions-source
============

Raw source code from Lions' Commentary on UNIX 6th Edition.

Derived from: http://www.tom-yam.or.jp/2238/src/

For Dennis Ritchie (1941 - 2011): ars longa vita brevis.

Reading entries:
- `m40.s`: entry point of boot process
- `sysent.c`: entries of system calls
- `conf.c`: char and block device drivers
- `low.s`: entry points of traps and interrupts
- Function `sched`: entry point of kernel proc swapper
